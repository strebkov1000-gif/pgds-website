### ШАГ 2: Задеплоить на Netlify (5 минут)

#### 2.1. Зарегистрироваться

1. Открыть: https://app.netlify.com/signup
2. Нажать **"GitHub"** (войти через ваш GitHub: strebkov1000-gif)
3. Разрешить доступ

#### 2.2. Создать сайт

1. **Add new site** → **Import an existing project**
2. **Deploy with GitHub**
3. Выбрать репозиторий: **pgds-website**
4. Настройки:
   ```
   Build command: npm run build
   Publish directory: out
   ```
5. **Deploy pgds-website**

⏱️ Подождать 2-3 минуты

✅ **Проверка:** Сайт открывается по Netlify URL

---

### ШАГ 3: Включить Netlify Identity (3 минуты)

#### 3.1. Enable Identity
1. Netlify dashboard → **Site settings**
2. **Identity** → **Enable Identity**

#### 3.2. Настроить Registration
1. **Registration preferences** → **Edit settings**
2. Выбрать **Invite only**
3. **Save**

#### 3.3. Enable Git Gateway
1. Прокрутить до **Services**
2. **Git Gateway** → **Enable Git Gateway**
3. **Save**

✅ **Проверка:** Git Gateway enabled

---
