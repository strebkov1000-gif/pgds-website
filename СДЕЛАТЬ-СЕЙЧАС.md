### ШАГ 1: Создать GitHub репозиторий (3 минуты)

#### 1.1. Создать репозиторий на GitHub

1. Открыть https://github.com/new
2. **Repository name:** `pgds-website` (или любое другое имя)
3. **Visibility:** Public (или Private если хотите)
4. **НЕ** ставить галочки на "Add README" или ".gitignore" (у нас уже есть)
5. Нажать **Create repository**

#### 1.2. Запушить код

**Скопируйте эти команды** (замените YOUR-USERNAME на ваш GitHub username):

```bash
cd "c:\Users\qtosh1\pgds-website"

git remote add origin https://github.com/YOUR-USERNAME/pgds-website.git

git push -u origin master
```

**Пример:**

### ШАГ 2: Задеплоить на Netlify (5 минут)

#### 2.1. Зарегистрироваться на Netlify

1. Открыть https://app.netlify.com/signup
2. Нажать **GitHub** (войти через GitHub аккаунт)
3. Разрешить доступ Netlify к GitHub

#### 2.2. Создать новый сайт

1. На главной странице Netlify нажать **Add new site**
2. Выбрать **Import an existing project**
3. Выбрать **Deploy with GitHub**
4. Найти и выбрать репозиторий **pgds-website**
5. Настройки сборки:
   ```
   Build command: npm run build
   Publish directory: out
   ```
6. Нажать **Deploy pgds-website**

⏱️ **Подождать 2-3 минуты** пока сайт соберётся

✅ **Проверка:** Появится URL типа `https://random-name-123.netlify.app`

---

### ШАГ 3: Включить Netlify Identity (3 минуты)

#### 3.1. Включить Identity

1. В Netlify dashboard → **Site settings** (вверху)
2. В левом меню → **Identity**
3. Нажать большую кнопку **Enable Identity**

#### 3.2. Настроить Registration

1. Прокрутить вниз до **Registration preferences**
2. Нажать **Edit settings**
3. Выбрать **Invite only**
4. Нажать **Save**

#### 3.3. Включить Git Gateway

1. Прокрутить вниз до **Services**
2. Найти **Git Gateway**
3. Нажать **Enable Git Gateway**
4. Выбрать роль: **Identity users** (по умолчанию)
5. Нажать **Save**

✅ **Проверка:** Git Gateway показывает статус "Enabled"

## 🔧 Если что-то пошло не так

### GitHub Actions упал

**Причина:** Неверный FTP пароль или сервер недоступен

**Решение:**
1. GitHub → Settings → Secrets → проверить FTP_PASSWORD
2. Проверить что Timeweb FTP доступен

### Админка не открывается

**Причина:** Не включен Identity или Git Gateway

**Решение:**
1. Netlify → Site Settings → Identity → Enable Identity
2. Services → Git Gateway → Enable

### Заказчик не может войти

**Причина:** Не приглашён через Identity

**Решение:**
1. Netlify → Identity → Invite users → ввести email заказчика
