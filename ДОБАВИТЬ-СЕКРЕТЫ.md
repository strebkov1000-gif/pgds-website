# 🔐 Добавление FTP секретов в GitHub

## Способ 1: Через веб-интерфейс (САМЫЙ ПРОСТОЙ - 2 минуты)

1. Открыть: https://github.com/strebkov1000-gif/pgds-website/settings/secrets/actions

2. Нажать **"New repository secret"** и добавить 3 секрета:

### Секрет #1
- **Name:** `FTP_SERVER`
- **Secret:** `vh434.timeweb.ru`
- Нажать **Add secret**

### Секрет #2
- **Name:** `FTP_USERNAME`
- **Secret:** `ca225020`
- Нажать **Add secret**

### Секрет #3
- **Name:** `FTP_PASSWORD`
- **Secret:** `Xz0n4@x$Vzhw`
- Нажать **Add secret**

✅ Готово! Теперь GitHub Actions сможет деплоить на Timeweb

---

## Способ 2: Через командную строку (если есть gh CLI)

```bash
gh secret set FTP_SERVER -b "vh434.timeweb.ru" -R strebkov1000-gif/pgds-website
gh secret set FTP_USERNAME -b "ca225020" -R strebkov1000-gif/pgds-website
gh secret set FTP_PASSWORD -b "Xz0n4@x$Vzhw" -R strebkov1000-gif/pgds-website
```
