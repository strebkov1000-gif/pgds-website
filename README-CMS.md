## ⚡ Быстрый старт для разработчика

### 1. Создать GitHub репозиторий

```bash
git remote add origin https://github.com/YOUR-USERNAME/pgds-website.git
git push -u origin master
```

### 2. Задеплоить на Netlify

1. Зайти на https://netlify.com
2. **New site from Git** → выбрать репозиторий
3. Build command: `npm run build`
4. Publish directory: `out`
5. Deploy

### 3. Включить Netlify Identity

1. **Site settings** → **Identity** → **Enable Identity**
2. **Services** → **Git Gateway** → **Enable Git Gateway**
3. **Registration** → **Invite only**

### 4. Настроить FTP для Timeweb (опционально)

Если нужен деплой на Timeweb:

1. GitHub → **Settings** → **Secrets** → **Actions**
2. Добавить секреты:
   - `FTP_SERVER`: vh434.timeweb.ru
   - `FTP_USERNAME`: ca225020
   - `FTP_PASSWORD`: [ваш пароль]

### 5. Пригласить заказчика

1. Netlify → **Identity** → **Invite users**
2. Ввести email заказчика
3. Отправить инструкцию: `ИНСТРУКЦИЯ-ДЛЯ-ЗАКАЗЧИКА.md`

## 📝 Как работает система?

```
Заказчик редактирует контент в админ-панели
          ↓
Изменения сохраняются в GitHub (через Git Gateway)
          ↓
GitHub Actions запускает сборку
          ↓
Статический сайт загружается на хостинг (Timeweb или Netlify)
          ↓
Изменения видны на сайте через 5-10 минут
```
