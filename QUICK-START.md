# ⚡ Быстрый старт - 5 шагов до готовности

Следуйте этим 5 шагам чтобы запустить CMS для заказчика за 20 минут.

---

## Шаг 1️⃣: GitHub (2 минуты)

```bash
# Создать репозиторий на GitHub.com, затем:
git remote add origin https://github.com/YOUR-USERNAME/pgds-website.git
git add .
git commit -m "Add Decap CMS"
git push -u origin master
```

✅ **Проверка:** Код появился на GitHub

---

## Шаг 2️⃣: Netlify (5 минут)

1. Зайти на **https://netlify.com**
2. **New site from Git** → выбрать репозиторий
3. Настройки:
   - **Build command:** `npm run build`
   - **Publish directory:** `out`
4. **Deploy site**

✅ **Проверка:** Сайт открывается по Netlify URL

---

## Шаг 3️⃣: Identity (3 минуты)

1. Netlify → Ваш сайт → **Site settings**
2. **Identity** → **Enable Identity**
3. **Registration preferences** → **Invite only**
4. **Services** → **Git Gateway** → **Enable Git Gateway**

✅ **Проверка:** Identity включен

---

## Шаг 4️⃣: Пригласить заказчика (2 минуты)

1. **Identity** → **Invite users**
2. Ввести **email заказчика**
3. **Send invite**

✅ **Проверка:** Заказчик получил письмо

---

1. GitHub → **Settings** → **Secrets and variables** → **Actions**
2. **New repository secret**:
   - Name: `FTP_SERVER`, Value: `vh434.timeweb.ru`
   - Name: `FTP_USERNAME`, Value: `ca225020`
   - Name: `FTP_PASSWORD`, Value: `[ваш FTP пароль]`
3. Сделать коммит → GitHub Actions задеплоит на Timeweb
