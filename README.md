# Сайт ПГДС (Проектирование и строительство газопроводов)

Корпоративный сайт компании ПГДС на Next.js с системой управления контентом.

## Технологии

- **Next.js 16** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **Lucide React** - иконки

## Структура проекта

```
pgds-website/
├── app/                    # Страницы приложения
│   ├── about/             # О компании
│   ├── services/          # Услуги
│   ├── projects/          # Проекты
│   ├── partners/          # Партнеры
│   ├── news/              # Новости
│   ├── vacancies/         # Вакансии
│   ├── tenders/           # Тендеры
│   ├── contacts/          # Контакты
│   └── admin/             # Админ-панель
├── components/            # React компоненты
│   ├── Header.tsx         # Шапка сайта
│   └── Footer.tsx         # Подвал сайта
├── data/                  # JSON файлы с данными
│   ├── content.json       # Основной контент
│   ├── projects.json      # Проекты
│   ├── news.json          # Новости
│   ├── vacancies.json     # Вакансии
│   └── tenders.json       # Тендеры
└── lib/                   # Утилиты
    └── utils.ts           # Вспомогательные функции
```

## Запуск проекта

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Сборка для production

```bash
npm run build
npm start
```

## Управление контентом

Все данные сайта хранятся в JSON файлах в папке `data/`. Для изменения контента:

1. Откройте нужный JSON файл в редакторе
2. Отредактируйте данные
3. Сохраните файл
4. Перезапустите сервер разработки

### Файлы данных:

- **data/content.json** - информация о компании, услуги, партнеры, регионы
- **data/projects.json** - список проектов
- **data/news.json** - новости компании
- **data/vacancies.json** - открытые вакансии
- **data/tenders.json** - активные тендеры

## Админ-панель

Доступна по адресу [http://localhost:3000/admin](http://localhost:3000/admin)

Панель предоставляет обзор контента и быстрый доступ к файлам данных.

## Цветовая схема

Сайт использует фирменные цвета компании:

- **Темно-синий (primary)**: `#003d7a` - основной цвет
- **Оранжевый (accent)**: `#ff8c00` - акцентный цвет

## Разделы сайта

1. **Главная** - обзор компании, статистика, услуги, карта присутствия
2. **О компании** - история, команда, сертификаты, техника
3. **Услуги** - подробное описание всех видов услуг
4. **Проекты** - портфолио реализованных проектов
5. **Партнеры** - список партнеров и клиентов
6. **Новости** - новости и события компании
7. **Вакансии** - открытые позиции и условия работы
8. **Тендеры** - текущие тендеры и закупки
9. **Контакты** - контактная информация, форма обратной связи

## Развертывание на хостинге

После завершения разработки проект можно развернуть на:

- **Vercel** (рекомендуется для Next.js)
- **Netlify**
- **Любой Node.js хостинг**

### Шаги для развертывания:

1. Создайте аккаунт на выбранной платформе
2. Подключите репозиторий (если используете Git)
3. Укажите команды сборки:
   - Build command: `npm run build`
   - Output directory: `.next`
4. Разверните проект
