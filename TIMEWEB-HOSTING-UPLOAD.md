# 🚀 Инструкция по загрузке сайта на Timeweb (обычный хостинг)

## ✅ Готово:
- Сайт собран в статические HTML файлы в папке `out/`
- Все файлы готовы к загрузке на хостинг

---

## Шаг 1: Получить FTP данные от Timeweb

### 1.1 Войдите в панель Timeweb
1. Зайдите на https://timeweb.cloud/
2. Войдите в личный кабинет
3. Перейдите в раздел **"Хостинг"** → ваш тариф **"Optimo"**

### 1.2 Найдите FTP доступы
В панели управления хостингом найдите:
- **FTP хост:** обычно `ftp.ваш-домен.ru` или `ca225020.tw1.ru`
- **FTP логин:** ваш логин (например, `ca225020`)
- **FTP пароль:** если забыли, можно сбросить в панели

📝 **Запишите эти данные!**

## Шаг 2: Вариант А - Через FileZilla (Рекомендуется)

### 2.1 Установить FileZilla
1. Скачайте FileZilla Client: https://filezilla.ru/get/
2. Установите программу

### 2.2 Подключиться к серверу
1. Откройте FileZilla
2. Заполните поля сверху:
   - **Хост:** `ftp://ca225020.tw1.ru` (или ваш FTP адрес)
   - **Имя пользователя:** `ca225020` (ваш логин)
   - **Пароль:** ваш FTP пароль
   - **Порт:** `21`
3. Нажмите **"Быстрое соединение"**

### 2.3 Найти папку для сайта
После подключения справа в FileZilla появятся папки сервера.

Найдите папку:
- `public_html` ИЛИ
- `www` ИЛИ
- `httpdocs`

Это корневая папка вашего сайта. **Откройте её.**

### 2.4 Очистить папку (если там есть файлы)
Если внутри есть старые файлы (например, `index.html` с заглушкой):
1. Выделите все файлы
2. Нажмите **Delete** (удалить)

### 2.5 Загрузить файлы сайта
1. **Слева** в FileZilla найдите папку: `C:\Users\qtosh1\pgds-website\out`
2. Откройте её (слева появятся все файлы вашего сайта)
3. **Выделите ВСЕ файлы и папки** внутри `out`:
   - Все HTML файлы (`index.html`, `about.html`, и т.д.)
   - Папку `_next`
   - Папку `images`
   - Файл `404.html`
   - Все остальные файлы
4. **Перетащите** их в **правую часть** FileZilla (в папку `public_html`)

⏳ Начнётся загрузка файлов. Дождитесь завершения (займёт 1-3 минуты).

## Шаг 3: Вариант Б - Через встроенный файловый менеджер Timeweb

### 3.3 Загрузить файлы
1. Нажмите кнопку **"Загрузить файлы"** или **"Upload"**
2. **Выберите ВСЕ файлы и папки** из `C:\Users\qtosh1\pgds-website\out`
3. Дождитесь завершения загрузки

⚠️ **Важно:** Загружайте содержимое папки `out`, а не саму папку!

## Шаг 5: Привязка домена (опционально)

Если хотите, чтобы сайт открывался по красивому адресу (например, `pgds-company.ru`):

### 5.1 Купить домен
- На Timeweb, REG.RU, или другом регистраторе
- Например: `pgds-company.ru`

### 5.2 Привязать домен к хостингу
**В панели Timeweb:**
1. Откройте раздел **"Домены"**
2. Нажмите **"Добавить домен"**
3. Введите ваш домен: `pgds-company.ru`
4. Выберите **"Привязать к хостингу"** → выберите ваш тариф Optimo
5. Сохраните

**В панели регистратора домена:**
1. Откройте управление DNS
2. Добавьте A-запись:
   ```
   Тип: A
   Имя: @
   Значение: IP адрес вашего хостинга (узнайте в панели Timeweb)
   TTL: 3600
   ```
3. Добавьте A-запись для www:
   ```
   Тип: A
   Имя: www
   Значение: IP адрес вашего хостинга
   TTL: 3600
   ```

⏳ DNS изменения применяются от 1 до 24 часов.

## 📋 Структура загруженных файлов

После загрузки в `public_html` должна быть такая структура:

```
public_html/
├── index.html           (главная страница)
├── about.html           (о компании)
├── contacts.html        (контакты)
├── news.html            (новости)
├── projects.html        (проекты)
├── services.html        (услуги)
├── tenders.html         (тендеры)
├── vacancies.html       (вакансии)
├── partners.html        (партнёры)
├── admin.html           (админ-панель - только для просмотра)
├── 404.html             (страница ошибки)
├── _next/               (папка с CSS/JS)
│   └── static/
│       └── css/
│           └── ...
└── images/              (папка с изображениями)
    └── ...
```

⚠️ **Важно:** Загружайте содержимое папки `out`, а не саму папку!

**Правильно:**
```
public_html/
├── index.html
├── about.html
└── _next/
```

**Неправильно:**
```
public_html/
└── out/
    ├── index.html
    ├── about.html
    └── _next/
```

## ⚠️ Важная информация

### Админ-панель НЕ работает
На обычном хостинге админ-панель (`/admin`) будет **только для просмотра**, так как:
- Нет Node.js для выполнения кода
- Нет базы данных для сохранения изменений

**Как редактировать сайт:**
1. Открыть `C:\Users\qtosh1\pgds-website\data\content.json` на вашем компьютере
2. Внести изменения
3. Пересобрать сайт: `npm run build`
4. Загрузить обновлённые файлы на хостинг

### Security Headers
На обычном хостинге security headers нужно настраивать через `.htaccess` файл (не через Next.js).

Если хотите добавить защиту, создайте файл `.htaccess` в `public_html`:
```apache
# Security Headers
<IfModule mod_headers.c>
    Header set X-Frame-Options "SAMEORIGIN"
    Header set X-Content-Type-Options "nosniff"
    Header set X-XSS-Protection "1; mode=block"
    Header set Referrer-Policy "strict-origin-when-cross-origin"
</IfModule>
```

## 🆘 Решение проблем

### Проблема: "403 Forbidden"
**Причина:** Нет файла `index.html` в корне
**Решение:** Убедитесь, что `index.html` находится в `public_html`, а не в подпапке

### Проблема: "404 Not Found" на всех страницах кроме главной
**Причина:** Сервер не настроен на обработку `.html` расширений
**Решение:** Ссылки работают так:
- `http://сайт.ru/about.html` ✅
- `http://сайт.ru/about` ❌ (может не работать)

### Проблема: Страницы без CSS/дизайна
**Причина:** Не загружена папка `_next`
**Решение:** Загрузите папку `_next` из `out/` на сервер в `public_html/_next/`

### Проблема: Не загружаются изображения
**Причина:** Не загружена папка `images`
**Решение:** Загрузите папку `images` из `out/` на сервер в `public_html/images/`
